Hello World

发表于 2020-03-13 分类于 杂谈 Disqus：
本文字数： 743

  大概花了四天时间，建立起了自己的第一个博客，虽然很简陋，但基本功能都实现了，勉强可以投入使用。建立期间也是走了不少弯路，最后选择了github pages + Hexo，毕竟免费还是香，不过这样也导致只能发挥普通博客的作用（也许还有其他用法，但目前技术水平有限）。markdown语法还不太熟，第一篇博文就当来练手了。

初衷

  做个人博客有几个原因吧，其一是可以记录自己学习的过程，有许多在实践过程中新学的知识没有经过系统的整理，最后要用的时候还得重新学习一遍，效率太低；其二是可以转移自身注意力（没办法闲着就想玩游戏），索性花时间来提升一下专注力，还能学点东西；其三是分享一下自己的想法或者学到的东西（虽然肯定达不到输出知识的水平，我能做的最多也就是搬运整合之类的工作），博客不是笔记本，它向全世界公开，欢迎每一个访客来这里交流自己的想法，若能对他人有所启发，本站的存在也就更有价值。

有待完善之处

  其实还挺多的，以后有时间再慢慢完善吧，有些问题挺棘手的，得从头开始（所以现在有点后悔没有选择NexT主题）。

  • 站内搜索（只搜索标题）
  • 标签系统（原作者做得很迷，没有发挥标签的作用）
  • 留言板
  • 图片lazy-load
  • 评论系统改进（目前需要翻墙才能看见）
  • 归档时间线
  • 各类统计
    • 个体：字数统计、阅读时间等
    • 全局：文章数量、标签数量、分类数量等
    • 站点：浏览次数、访客数量等
  • 目录无法随页面滚动，分级目录间隔过大，字数过多时不会自动换行

结语

  听过一句话：生命不息，折腾不止。虽然现在只能算个技术小白，要学的东西还很多，但既然起了头，就不能半途而废，若能养成写博的习惯，一直坚持下去，我现在埋下的这颗小小的种子，在未来也能结出累累硕果吧！

《海猫》游玩有感

发表于 2020-02-16 更新于 2020-03-13 分类于 杂谈 Disqus：
本文字数： 12k
--谨以此故事，献给吾最爱的魔女贝阿朵莉切


前言

接触海猫的过程

  首次接触海猫应该是在高一寒假，当时是奔着悬疑推理去的，然而由于动画版质量不高，看到最后也只留下了恐怖猎奇的印象。不过不得不说，海猫的背景设定是相当对我口味的:暴风雪山庄模式，西洋风格的人物设计等…因此很早就下定决心要推完海猫原作。当时的动力只是解开前四个ep的谜题而已，毕竟我完全想不出如何破解那一大堆密室 所以其实我早就是个剧情党了，中途因为一直没找到合适的时间，耽搁了很久，正式开始推海猫原作是在大二上，断断续续花了一个多月的时间推完了。虽然不是一气呵成，导致有点玩后面忘前面，但空出了沉淀的时间，让我更多地去思考作者想要传达的东西，心态也由原本的追求 How done it 转变为了追求 Why done it ，即从“推理党”变成了“剧情党”。即便是写下这篇博客的现在，有些谜题依然徘徊在我的脑海里，毕竟07自己在某些问题上也没有给出一个明确的答复。我与海猫的缘分不会到此为止，等看完漫画，推完HD版再回来补充这篇博客，给海猫的故事画上一个完满的句号。
